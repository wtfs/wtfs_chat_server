wtfs professional chitchat

an open source video chat/conference solution, no extra plugins or extensions are required

About

wtfs chitchat is an opensource video chat and video conference service. wtfs chitchat uses the new HTML5 web standard Webrtc , which is available on most modern browsers.

In addition to Video and Audio streaming, wtfs chitchat provides a pluginfree screen sharing(screensharing) option, this option is only available on Chromium and Google Chrome.

All client site executed code is written in plain JavaScript without CoffeeScript or other auto generated code. The Server is written in Erlang OTP.

Both the client JavaScript Souce Code and the Server Erlang Souce Code is available on github.
